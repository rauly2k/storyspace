StorySpace is a mobile application built with Flutter that brings stories to life for children and parents. It leverages AI to create personalized tales, generate beautiful illustrations, and even place your child right into the story.

## âœ¨ Features

The app is being built in phases, with the following core features planned:

- **AI Story Generation**: Create unique stories from templates or based on user prompts.
- **Personalization**: Put your child in the story by using their name and a photo for style-transferred illustrations.
- **Curated Library**: Browse a library of pre-made stories across various categories.
- **Image Generation**: Each story is accompanied by AI-generated images for the cover and scenes.
- **Offline Access**: Download stories to read anytime, anywhere.
- **Audio Narration**: Listen to stories with text-to-speech narration.
- **Parent & Kid Profiles**: Manage multiple kid profiles under a single parent account.

## Getting Started

This project is a starting point for a Flutter application.

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK: [Installation Guide](https://docs.flutter.dev/get-started/install)
- A configured IDE like VS Code or Android Studio.

### Installation

1. Clone the repo: `git clone https://github.com/your-username/storyspace.git`
2. Navigate to the project directory: `cd storyspace`
3. Install dependencies: `flutter pub get`
4. Run the app: `flutter run`
