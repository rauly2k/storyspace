2. **onboarding_slide2_image.png**

## Notes:
- Images will be displayed with a custom wave-shaped bottom edge using the `OnboardingWaveClipper`
- Format: PNG with transparency support recommended
- The images should work well with the app's color scheme (primary, secondary colors)
- If images are missing, the app will show a gradient background with icons as fallback
